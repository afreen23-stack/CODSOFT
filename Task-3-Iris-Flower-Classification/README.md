# Iris Flower Classification

## Objective

The goal of this project is to build a machine learning model that can classify Iris flowers into three different species based on their measurements.

## Dataset

The Iris dataset contains 150 records with the following features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species (Target Variable)

The three species included in the dataset are:

* Iris-setosa
* Iris-versicolor
* Iris-virginica

## Steps Performed

1. Imported necessary Python libraries.
2. Loaded the Iris dataset.
3. Explored the dataset using statistical summaries.
4. Selected features and target variables.
5. Split the dataset into training and testing sets.
6. Trained a machine learning classification model.
7. Evaluated the model using accuracy and classification report.
8. Visualized performance using a confusion matrix.

## Model Used

Random Forest Classifier

## Evaluation Metrics

* Accuracy Score
* Classification Report
* Confusion Matrix

## Result

The model achieved an accuracy of approximately **100%**, correctly classifying the Iris flower species based on the given measurements.

## Conclusion

Machine learning models can effectively classify Iris flower species using sepal and petal measurements. The Random Forest classifier performed very well on this dataset and achieved excellent prediction accuracy.
