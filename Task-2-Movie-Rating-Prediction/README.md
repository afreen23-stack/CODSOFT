# Movie Rating Prediction with Python

## Objective

Build a machine learning model to predict movie ratings using features such as year, duration, genre, and other movie information.

## Dataset

IMDb Indian Movies dataset containing movie details such as name, year, duration, genre, director, actors, votes, and ratings.

## Steps Performed

* Loaded and explored the dataset
* Handled missing values
* Cleaned Year and Duration columns
* Performed Exploratory Data Analysis (EDA)
* Visualized trends using graphs
* Built a Linear Regression model to predict movie ratings

## Analysis Performed

* Year with highest average rating
* Relationship between movie duration and rating
* Top 10 movies based on rating
* Directors with the most movies
* Number of movies released each year

## Model Used

Linear Regression

## Evaluation Metric

Mean Squared Error (MSE)

## Result

The model predicted movie ratings with a Mean Squared Error of approximately **1.83**.

## Conclusion

Movie rating prediction can be performed using regression techniques, and exploratory data analysis helps identify patterns and trends in movie datasets.
